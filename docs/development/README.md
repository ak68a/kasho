## Getting Started

### Prerequisites
- Go 1.21 or later
- Node.js 18 or later
- Docker
- PostgreSQL
- Redis
- Make

### Quick Start
1. Clone the repository
2. Run `make setup` to set up the development environment
3. Run `make dev` to start the development servers
4. Visit `http://localhost:3000` for the web application
